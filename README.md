# Portfolio
## Introduction
개인 포트폴리오 사이트입니다.

GSAP을 통해 인터랙션을 구현했으며, CSS 전처리기 학습을 위해 SCSS를 사용했습니다.

## Stack
- Next.js @14.2.15
- GSAP @3.12.5
- Lenis @1.1.14
- SCSS @1.80.3
- clsx @2.1.1
- highlight.js @11.11.0
- pnpm

## Installation & Run
```bash
pnpm install

pnpm dev
```

## Deployment & Domain

배포는 Vercel을 통해 배포했으며 도메인은 Route 53을 통해 구매했습니다.
